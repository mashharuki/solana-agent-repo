# Product Overview

Solana Agent Demo is a chat-first application for Solana BootCamp sessions. It helps users perform blockchain tasks from natural language while keeping wallet control and transaction signing in the user's hands.

## Core Capabilities

- Wallet-gated chat experience: users must connect a wallet before interacting.
- Natural-language blockchain assistance: agent responses explain Solana operations in clear Japanese.
- Transaction construction workflow: the agent prepares transaction payloads for user review and signature.
- On-chain context visibility: users can see wallet context such as SOL balance and NFT ownership.
- Devnet-first operation: all blockchain interactions are designed for safe demo and learning environments.

## Target Use Cases

- Instructor-led Solana BootCamp demos.
- Learner exploration of Solana operations without deep protocol knowledge.
- Interactive explanation of transaction outcomes after user-approved signing.

## Value Proposition

- Converts complicated Solana operations into guided conversation.
- Enforces a safe trust model: agent builds, user signs.
- Combines practical wallet UX, AI guidance, and on-chain actions in one coherent demo surface.
